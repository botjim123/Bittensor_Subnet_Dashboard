Index Methodology
Index Family 8 indexes — 7 BSACS sector indexes + Total Market (BSACS-TM)
Base Date September 4, 2025 — base value 1000 for all indexes
Weighting Market cap weighted. Daily weights = prior-day market cap / total sector market cap
Formula Index(t) = Index(t−1) × Σ( weight(i) × price(i,t) / price(i,t−1) )
Rebalancing Continuous daily. Weights recalculate every day using prior-day market caps
New Entrant Rule Weight = 0 on first appearance. Enters fully from day 2. Prevents distortion from new constituents
Eligibility Must have verified public description. Subnets without documentation flagged Unclassified and excluded from sector indexes
Exclusions SN0 (Root network) excluded — not an investable subnet
Price Data USD price from Taostats API pool endpoint — AMM price from on-chain liquidity pool × TAO/USD rate
Classification BSACS taxonomy — 3 tiers: Sector → Industry Group → Industry. Modeled on CoinDesk DACS. Primary classification only for index purposes
DACS Analogy DACS classifies TAO at the protocol layer. BSACS classifies subnets at the application layer — additive, not competing
Data Source Taostats API (api.taostats.io) — descriptions from github.com/taostat/subnets-infos
